# Teoria dos Sistemas de Niklas Luhmann

## Visão Geral

Um repositório educacional sobre a Teoria dos Sistemas Sociais de Niklas Luhmann, um dos sociólogos mais influentes do século XX.

## Conteúdo

- Introdução à Teoria dos Sistemas
- Conceitos Fundamentais
- Aplicações na Educação
- Estudos de Caso
- Recursos Adicionais

## Módulos

1. **Introdução** - Apresentação geral da teoria
2. **Conceitos Básicos** - Autopoiese, diferenciação sistêmica, comunicação
3. **Sistemas Sociais** - Estrutura e funcionamento
4. **Aplicações Práticas** - Educação, organizações, sociedade
