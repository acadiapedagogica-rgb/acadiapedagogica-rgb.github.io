Aplicações da Teoria

Aplicações Práticas dos Sistemas Sociais de Luhmann

Início Conceitos Módulos Aplicações Recursos

Educação

O sistema educacional opera com o código binário saber/não saber.

Código: saber / não saber

Direito

O sistema jurídico opera de forma fechada, tomando decisões com base no código lícito/ilícito.

Código: lícito / ilícito

Economia

O sistema econômico se organiza em torno do pagamento/não pagamento.

Código: pagar / não pagar

Ciência

O sistema científico busca a verdade e opera com o código verdadeiro/falso.

Código: verdadeiro / falso

Política

O sistema político opera com o código governo/oposição.

Código: governo / oposição

Mídia

O sistema da mídia opera com o código informação/não informação.

Código: informação / não informação
