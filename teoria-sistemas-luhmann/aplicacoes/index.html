Aplicacoes da Teoria

Aplicacoes Praticas dos Sistemas Sociais de Luhmann

Inicio Conceitos Modulos Aplicacoes Recursos

Educacao

O sistema educacional opera com o codigo binario saber/nao saber. A teoria de Luhmann ajuda a entender como a educacao se diferencia de outros subsistemas e como ela se comunica internamente.

Codigo: saber / nao saber

Direito

O sistema juridico opera de forma fechada, tomando decisoes com base no codigo licito/ilicito. A analise luhmanniana revela como o direito se autopoie e como se diferencia dos demais subsistemas sociais.

Codigo: licito / ilicito

Economia

O sistema economico se organiza em torno do pagamento/nao pagamento. Luhmann analisa como o dinheiro funciona como meio de comunicacao e como a economia se diferencia dos demais sistemas.

Codigo: pagar / nao pagar

Ciencia

O sistema cientifico busca a verdade e opera com o codigo verdadeiro/falso. A teoria ajuda a entender o processo de avaliacao de pesquisas, publicacoes e a dinamica de producao de conhecimento.

Codigo: verdadeiro / falso

Politica

O sistema politico opera com o codigo governo/oposicao. Luhmann analisa como a politica toma decisoes coletivamente vinculantes e como se diferencia dos outros sistemas sociais.

Codigo: governo / oposicao

Midia

O sistema da midia opera com o codigo informacao/nao informacao. Luhmann examina como a midia seleciona, processa e dissemina informacoes, influenciando a percepcao social da realidade.

Codigo: informacao / nao informacao
